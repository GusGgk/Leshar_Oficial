Adicionando autor na tabela habilidade:

1. ALTER TABLE habilidade ADD COLUMN autor varchar(100);
2. ...novo.php -> adiciona o campo na tabela ($autor = $_POST['autor'] ?? '';) -> adiciona na $stmt = conexao e $stmt->bind_param (não esquecer dos ?,?,? e dos ssss
3. ...novo.js -> adiciona no var e no fd.append copiando a linha de cima
4. ...novo.html -> copia a div do input de cima e muda os atributos
5. ...(habilidade).js -> dentro da function carregarLista() adiciona na table e embaixo na <td>${objeto.(autor)}</td>
6. ...get.php(OPCIONAL) -> se o crud for de habilidades ou perfil, colocar o atributo nos dois $stmt = $conexao->prepare
